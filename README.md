# **Fitness Coach Chat Application**

Fitness Coach is an AI-powered chat application that allows users to interact with a personalized AI fitness coach. This application provides customized fitness advice, workout plans, and motivation based on user goals and preferences.

### **Features:**

- User-friendly chat interface
- Personalized fitness advice
- Detailed workout plans
- Motivation and support

### **Prerequisites:**

Before you begin, ensure you have the following installed:

- Anaconda or Miniconda
- Python 3.11

### **Setup Instructions:**

1. Create a Conda Environment:
   - Run the following command to create a new Conda environment named `fitnessenv` with Python 3.11:
     ```
     conda create -n fitnessenv python==3.11 -y
     ```

2. Activate the Environment:
   - Activate the newly created environment with:
     ```
     conda activate fitnessenv
     ```

3. Install Requirements:
   - Install the necessary dependencies using pip:
     ```
     pip install -r requirements.txt
     ```

4. Run the Application:
   - Start the application by running:
     ```
     python app.py
     ```

### **Usage:**

Once the application is running, open your web browser and navigate to `http://127.0.0.1:5000/` to access the chat interface. You can start asking questions, and the AI will respond based on your input.
